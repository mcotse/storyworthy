- **FAB (Floating Action Button):** Circular save button positioned above nav bar
- **Entry Cards:** Collapsed view showing date and preview text
- **Search Bar:** Sticky at top, expandable with cancel option
- **Settings Sections:** Grouped cards with toggles, dropdowns, and action buttons
